В данной работе содержатся:
- Тест-Сьюты в папке "Test-Suites"
- Фреймворк и автотесты написанные на языке Java (в папке "Bank")

Для запуска автотестов необходимо:
- скачать драйвер - "chromedriver"
- установить путь до вашего драйвера в переменной "chromeDriverPath" в классе "Resources.java" (класс находится -  \\Bank\src\test\java\resources\Resources.java)
